### 다음 중 각 함수를 호출했을 때, 예상되는 출력값을 보기에서 고르시오.   
(실제로 세 함수를 동시에 호출하면 Error가 생기는 함수 이후로는 작동하지 않습니다. 문제 의도는 각 함수의 출력값이라는 점을 참고바랍니다.)
```js
test1();
test2();
test3();

function test1() {
	console.log('javascript');
}

var test2 = function() {
	console.log('javascript');
}

let test3 = function() {
	console.log('javascript');
}
```

```
(2) : TypeError
(3) : ReferenceError
```
