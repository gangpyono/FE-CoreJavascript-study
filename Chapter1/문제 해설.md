## 📌문제5
### 다음 (1)의 결과물을 예측해보세요.
```js
  const obj1 = {
        a: 123,
        b: [1, 2, 3],
      };

      const f1 = (ob) => {
        const res = { ...ob };
        res.b[1] = 1234;
        return res;
      };

      const obj2 = f1(obj1);
      console.log(obj1.b[1] === obj2.b[1]); // (1)
```
## 답안 작성
```
(1) : true
해설 : spread연산자는 바로 아래 단계 값만 복사를 하기떄문에, 참조형 데이터인 obj1.b에대한 복사가 이루어질땐 데이터가아닌 주소값이 복사된다.
      따라서 obj.b와 ob2.b는 동일한 배열을 가리키게됨으로 obj2.b에 접근하여 값을 변경하게되었을때 obj1.b의 값또한 변경되는 현상이 발생되어 true를 반환하게된다.
```
