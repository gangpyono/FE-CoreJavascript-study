## 📌문제3
### 다음 코드의 출력 결과로 알맞은 요소(`*`)를 보기에서 고르시오.
### 보기
```
요소1 : empty (실제로 값이 없고 순회대상에서 제외되는 것)
요소2 : 'two'
요소3 : 'three'
```
### 코드
```js
const arr = [];

arr[2] = 'two';
console.log(arr);  // (1)

arr.length = 5;
arr[3] = 'three';
console.log(arr);  // (2)

arr.length = 2;
console.log(arr);  // (3)

arr.length = 6;
console.log(arr);  // (4)
```
